# DS-Forge

DS-Forge is an AI-powered platform for practicing real-world
data analysis and data science problems using datasets,
SQL, Python, and Pandas.

## Features

- Real dataset challenges
- SQL + Python coding environment
- AI generated questions
- AI hints and explanations
- Interview-style problems

## Vision

To become the leading AI-powered platform for mastering
data analysis and data science interview skills.
