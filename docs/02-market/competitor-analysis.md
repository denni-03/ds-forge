# Competitor Analysis

## LeetCode
Strengths
- large problem library
- strong coding interview focus

Weaknesses
- little focus on data analysis
- problems mostly algorithmic

## Kaggle
Strengths
- real datasets
- machine learning competitions

Weaknesses
- steep learning curve
- not designed for structured practice

## DataCamp
Strengths
- structured courses
- beginner friendly

Weaknesses
- limited problem-solving practice

## StrataScratch
Strengths
- SQL interview questions
- real company examples

Weaknesses
- limited Python practice
- limited datasets

## Opportunity
Combine the strengths of these platforms into a single environment focused on data practice.
