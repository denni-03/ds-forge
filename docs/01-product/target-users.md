# Target Users

## Primary Users

### 1. Data Analyst Job Seekers
Individuals preparing for data analyst interviews who need to practice SQL and data analysis problems.

Needs:
- SQL query practice
- dataset analysis
- interview preparation

### 2. Data Science Students
Students learning Python, Pandas, and machine learning.

Needs:
- hands-on dataset practice
- problem-based learning
- real-world analysis experience

### 3. Early Career Data Professionals
Professionals transitioning into data roles.

Needs:
- improve analytical skills
- prepare for technical interviews
- practice real scenarios

## Secondary Users

### Educators
Teachers who want structured datasets and exercises for students.

### Hiring Managers
Companies that want to evaluate candidate data skills.

## User Goals

Users want to:

- practice realistic data problems
- improve SQL and Python skills
- prepare for interviews
- simulate real job tasks
